# janken soul 拳魂
## version
- Python 3.9.13
- mediapipe 0.10.14
- opencv-python 4.6.0
## 功能
- 一局猜拳
- 必定會贏的猜拳
- 必定會輸的猜拳
- 可以連續3局的猜拳
